Project Overview

This project implements an intelligent cybersecurity attack detection system designed to identify **Byzantine and anomalous cyber attacks** using Machine Learning, Ensemble Models, and Deep Learning techniques.

The system analyzes network and system behavior data to classify malicious activities and detect abnormal patterns that may indicate cyber threats.

This project demonstrates a complete **end-to-end ML pipeline**, including:

* Data preprocessing & feature engineering
* Model training and evaluation
* Ensemble learning
* Deep Learning (RNN-based detection)
* Model deployment using Flask
* Visualization and performance analysis

---

 Objectives

* Detect cyber attacks using advanced ML algorithms
* Improve detection accuracy using ensemble learning
* Capture sequential attack behavior using RNN models
* Provide a deployable cybersecurity detection system
